# 新增 .h 头文件注释补全

## 注释规范（对齐现有代码）

参考 [`LoginGatewayRegistry.h`](LoginServer/LoginGatewayRegistry.h)、[`LoginServer.h`](LoginServer/LoginServer.h)：

- 文件头：`@file` + 一行 `@brief`；复杂模块可加「职责 / 消息 / 调用方」短段落（不用 `@` 标签嵌套）
- 类/结构体：`/** @brief ... */`；public 方法写 `@brief`，必要时 `@param` / `@return`
- 成员变量：行尾 `/**< ... */`
- 自由函数：每个导出函数一行 `/** @brief ... */`

**范围**：仅注释，不调整接口、不新建 .md。
